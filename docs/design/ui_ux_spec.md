# MG-0003: Pixel Mercenary UI/UX 스펙

> **게임명**: Pixel Mercenary (픽셀 용병단)
> **장르**: 턴제 전술 RPG
> **카테고리**: Year 1 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 턴제 전투 UI 배치 용이
- 캐릭터 정보 + 전투 영역 세로 분할
- 스킬 선택 UI 하단 배치

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 캐릭터 선택 | 캐릭터 영역 | 탭 |
| 커맨드 선택 | 80×56dp | 탭 |
| 스킬 선택 | 스킬 팝업 | 탭 |
| 적 타겟 선택 | 적 영역 | 탭 |
| 정보 확인 | 캐릭터/적 | 롱프레스 |

### 터치 피드백
- 캐릭터 선택: 외곽선 강조
- 커맨드 선택: 버튼 확대 + 색상 변화
- 타겟 선택: 타겟 마커 표시
- 공격 실행: 슬래시 이펙트

---

## 5. 정보 계층

| 레벨 | 요소 | 상시 표시 |
|------|------|----------|
| L1 (상시) | 턴, HP, 상태 | ✓ |
| L2 (상호작용) | 캐릭터 상세 정보 | 롱프레스 시 |
| L3 (팝업) | 스킬 목록, 아이템 목록 | 커맨드 선택 시 |
| L4 (모달) | 전투 결과, 설정 | 이벤트 시 |
