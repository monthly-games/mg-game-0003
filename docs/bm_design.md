## 3. 상세 BM 설계

### 3.1 영웅/용병 수집 BM

#### 무료 구조
- 스토리 클리어로 기본 용병 8종 획득
- 일일 무료 뽑기 1회
- 던전/보스 드롭으로 용병 조각 파밍
- SR급 15종 이상 무료 수집 가능

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 일반 뽑기 | 10연차 1회 | 프리미엄 재화 2,800 |
| 픽업 배너 | 신규/한정 용병 확률 UP | 동일 |
| 천장 시스템 | 100연차 SSR 확정 | - |
| 선택권 패키지 | 10연차 + SSR 선택권 | $19.99 |
| 용병 스킨 | 외형 변경 (성능 동일) | $4.99~$9.99 |

### 3.2 성장/강화 BM

#### 무료 구조
- 방치 전투로 경험치/골드 자동 획득
- 일일 던전에서 강화 재료 파밍
- 본부/연구소 업그레이드 무료

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 스타터 패키지 | 레벨업 재화 대량 | $4.99 |
| 성장 가속 패스 | 경험치/골드 +100% (30일) | $9.99/월 |
| 장비 강화 패키지 | 강화석 + 장비 세트 | $9.99 |
| 본부 부스트 | 건설 시간 단축 | 프리미엄 재화 |

### 3.3 방치 수익 BM

#### 무료 구조
- 오프라인 수익 상한: 10시간
- 전투 로그/하이라이트 리포트 무료

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 방치 수익 2배 | 광고 시청 시 적용 | 무료 (광고) |
| 방치 수익 3배 패스 | 상시 적용 (30일) | $4.99/월 |
| 오프라인 확장 | 24시간까지 누적 | 패스 포함 |
| 퀵 배틀 티켓 | 2시간 즉시 보상 | 프리미엄 재화 50 |

- 길드 가입/활동 무료
- 주간 랭킹 보상 지급

## 6. BM 밸런스 원칙

### P2W 방지선
- 무과금도 SSR 용병 3~5명 확보 가능 (3개월 기준)
- PVP 상위 20%는 무과금 도달 가능
- 시너지/빌드가 스펙보다 중요하게 설계

### 과금 압박 최소화
- 방치 시스템으로 플레이 시간 강제 없음
- 콘텐츠 진입 장벽은 과금이 아닌 플레이 시간
- 천장 시스템으로 뽑기 좌절 방지
