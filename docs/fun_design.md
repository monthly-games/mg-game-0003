## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 방치 전투로 자원/장비 획득                           │
│         ↓                                               │
│  2. 용병단(영웅, 장비, 스킬) 강화                        │
│         ↓                                               │
│  3. 던전/보스/탐험 진행                                  │
│         ↓                                               │
│  4. 기지/본부 업그레이드 ─────────────→ (1로 반복)       │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **방치 전투**: 자동 전투 시스템으로 지속적인 자원/장비 획득
2. **용병단 강화**: 영웅 레벨업, 장비 강화, 스킬 해금 등 성장 투자
3. **던전/보스/탐험**: 성장한 전력으로 새로운 콘텐츠 도전
4. **기지/본부 업그레이드**: 용병단 전체에 영향을 주는 메타 성장

---

## 3. 재미 축 (Fun Pillars)

### 3.1 성장/수집 (Growth)
- **목표**: 용병단이 지속적으로 강해지는 체감 제공
- **핵심 경험**: 내 용병들이 점점 강해져서 뿌듯함
- **주요 요소**:
  - 영웅 레벨/승급 시스템
  - 장비 강화/세트 효과
  - 스킬 트리 해금
  - 도감 수집 보상

### 3.2 빌드/시너지 (Build)
- **목표**: 파티 구성 자체를 전략적 퍼즐로 만들기
- **핵심 경험**: 최적의 조합을 찾아냈을 때의 성취감
- **주요 요소**:
  - 역할(탱/딜/힐/버프) 시스템
  - 종족/직업 시너지
  - 세트 장비 효과
  - 스킬 콤보

### 3.3 파훼/도전 (Challenge)
- **목표**: 강력한 적을 전략으로 돌파하는 재미
- **핵심 경험**: 막혔던 보스를 빌드 변경으로 클리어했을 때 쾌감
- **주요 요소**:
  - 보스 패턴/약점 시스템
  - 속성 상성
  - 제한 조건 던전
  - 랭킹 도전

## 4. 디자인 레버 (Design Levers)

### 4.1 파티 구성 퍼즐화
| 레버 | 설명 |
|------|------|
| 역할 명확화 + 시너지 | 탱/딜/힐/버프 역할과 세트/종족/직업 시너지로 구성 자체를 퍼즐화 |
| 구현 방향 | 엘프 3인 시너지(치명타+10%), 기사단 시너지(방어+15%) 등 |

### 4.2 방치 보상 리포트
| 레버 | 설명 |
|------|------|
| 전투 로그/하이라이트 | 오프라인 보상을 단순 숫자가 아닌 전투 리포트로 제공 |
| 구현 방향 | "용사 렉스가 드래곤을 처치! MVP 선정", 하이라이트 컷신 |

### 4.3 관전 재미 요소
| 레버 | 설명 |
|------|------|
| 웨이브 중간 목표 | 노데스, 시간 제한 등 관전 유저에게도 재미 부여 |
| 구현 방향 | "30초 내 클리어 시 보너스 골드", "무피해 클리어 시 장비 드롭률 UP" |

### 4.4 평행 성장 축
| 레버 | 설명 |
|------|------|
| 본부/연구 트리 | 레벨/장비 외에 용병단 전체에 영향을 주는 성장 축 |
| 구현 방향 | 본부 레벨 → 영웅 최대 레벨 상한, 연구소 → 패시브 버프 영구 적용 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 일일 접속 횟수 | 4회 이상 | 방치 보상 수령 패턴 분석 |
| 파티 변경 빈도 | 주 3회 이상 | 파티 편성 변경 로그 |
| 보스 도전률 | 80% 이상 | 언락된 보스 대비 도전 비율 |
| 영웅 수집률 | 월 70% | 획득 가능 영웅 대비 보유율 |
| D7 리텐션 | 35% 이상 | 7일차 잔존율 |
