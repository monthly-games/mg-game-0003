## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 코어 성장 루프/방치 수익 밸런스 튜닝 |
| 7~10주 | 초반 용병/장비 풀 정리 |
| 11~12주 | 첫 시즌 용병 배너 준비 |

**핵심 과제:**
- 방치 수익 밸런스 최적화
- 용병 성장 곡선 안정화
- 튜토리얼 개선

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 신규 용병 시즌 1차 (배너/이벤트) |
| 5개월 | 던전 난이도 단계 추가 |
| 6개월 | 신규 용병 시즌 2차 + 보스 세트 |
| 7개월 | "오프라인 리포트" UX 개선 |
| 8개월 | 신규 용병 시즌 3차 + 여름 이벤트 |

**핵심 과제:**
- 월 1~2회 배너/이벤트 사이클
- 던전 콘텐츠 확장
- 오프라인 경험 개선

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 클랜/파티 기반 보스 도입 (0019 연계 전초전) |
| 10개월 | 전생/심화 콘텐츠 추가 |
| 11개월 | 시즌 패스 도입 |
| 12개월 | 연말 레이드(0012) 연동 + 1주년 |

**핵심 과제:**
- 길드 콘텐츠 기반 마련
- 장기 유저용 엔드콘텐츠
- 크로스 게임 연동 검증
